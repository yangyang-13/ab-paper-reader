# v2.1 更新说明 - 价值分排序功能

## 🎯 新增功能

### 论文排序机制

增加了按照平台价值评分排序的功能，帮助用户快速找到最有价值的论文。

## ✨ 功能特性

### 1. 排序选项

在控制面板中新增"排序"下拉框，提供两种排序方式：

- **按日期**（默认）：按论文发布日期降序排列
- **按价值分**：按平台价值评分从高到低排列

### 2. 智能排序

当选择"按价值分"排序时：
- 自动提取每篇论文的平台价值评分
- 按照评分从高到低（100 → 0）排序
- 没有评分的论文排在最后

### 3. 使用场景

**按日期排序**：
- 查看最新发布的论文
- 追踪学术研究前沿
- 每日例行浏览

**按价值分排序**：
- 快速找到最有价值的论文
- 优先阅读高分论文
- 节省筛选时间

## 📊 使用方法

### 步骤1：选择排序方式

在页面顶部的控制面板中：
```
排序：[按日期 ▼]  →  选择  [按价值分 ▼]
```

### 步骤2：点击查询

点击"查询"按钮，系统会按照选择的方式重新排序论文列表。

### 步骤3：查看结果

- 论文卡片会按照评分从高到低显示
- 每篇论文右上角可以看到价值评分标识
- 高价值论文（70+分）会显示绿色标识

## 🎨 视觉效果

价值评分标识：
- 🟢 **高价值** (70-100分)：绿色标识
- 🟠 **中等价值** (40-69分)：橙色标识
- ⚪ **低价值** (0-39分)：灰色标识

## 🔧 技术实现

### 后端API

新增 `sort_by` 参数：

```http
GET /api/papers?sort_by=value&limit=100
```

参数说明：
- `sort_by=date`：按日期排序（默认）
- `sort_by=value`：按价值分排序

## ⚙️ 配置说明

### 默认排序

系统默认使用"按日期"排序，确保用户首先看到最新的论文。

### 排序持久化

当前排序选择不会持久化，刷新页面后会恢复为默认的"按日期"排序。
